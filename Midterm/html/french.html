tâche 1

Pour cette section, vous devrez changer le bouton "tâche 1" dans le menu déroulant. Chaque fois que vous cliquez dessus, un nouveau fichier HTML (table.html) contenant les informations suivantes devrait être ouvert: Créez une table avec vos films préférés. Vous devez avoir au moins 4 en-têtes de tableau, vous pouvez utiliser par exemple: Nom, Directeur, Date de publication et Caractère principal. Remplissez ce tableau avec au moins 3 films.

Table.html

Dans ce fichier Table.html, vous devez créer un menu avec deux boutons: Résumé et Trailer avec des liens vers la même page à l'aide de l'id et un bouton Home pour revenir à index.html.

Sommaire

For the first movie of the list you should create a briefly text about the movie an include an image with Alt and Title attributes

bande annonce

Create a section Trailer on your Table.html and include a video player from youtube with a trailer related to your first movie of the list. If you dont find a related trailer, insert a random Youtube Video.

domicile

The Home button should take you back to this page.

tâche 2

La tâche 2 consiste à créer un menu déroulant sur la barre de navigation avec la langue du titre. Si vous cliquez sur le bouton, vous devriez voir deux petits drapeaux, l'un devrait être le drapeau du Canada et l'autre, le drapeau de votre pays. Si je clique sur votre drapeau de pays, je souhaite voir cette page Web dans votre langue maternelle et le drapeau du Canada devrait le replacer en anglais.
